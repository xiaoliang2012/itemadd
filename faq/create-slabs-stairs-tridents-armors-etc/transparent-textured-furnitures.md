# 如何创建半透明材质的家具

{% hint style="warning" %}
如果您想制作一些 透明/半透明 的材质, 必须确保你的物品是基于一个透明的原版物品.(如：玻璃板)
这里举个栗子:
{% endhint %}

```yaml
resource:
  material: WHITE_STAINED_GLASS
  generate: false
  model_path: item/your_model
```
