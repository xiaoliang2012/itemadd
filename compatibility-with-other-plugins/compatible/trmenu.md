```yaml
material: 'source:ITEMADDER:itemId'

```

### 如何在菜单中使用自定义背景（fontimage）

```yaml
Title: '%img_offset_-16%%img_blank_menu%'
```

{% hint style="warning" %}
PlaceholderAPI 为必要前置
{% endhint %}

`%img_offset_-16%` 的含义为：将图像 `%img_blank_menu%` 向后偏移 **16** 个像素.
