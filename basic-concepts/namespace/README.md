# Namespace（命名空间）

## Namespace（命名空间）

你可以发现 Itemsadder 使用 **命名空间** 来识别并管理其中的内容.\
其实你也可以将 **命名空间** 理解为 **分组** ，一个 **命名空间** 代表一个 **组** 其中包含 **物品**/**方块**/**怪物**....\
**命名空间** 可以使你更方便快捷的管理 **物品**/**方块**/**怪物**... 的配置

### 栗子

所有 **realcraft** 的物品都在 **realcraft** 的命名空间中，所以当你使用 `/iaget` 时你可以看到所有的物品id以 `realcraft:` 为前缀
